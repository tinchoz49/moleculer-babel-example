# moleculer-babel-example

## Build Setup

``` bash
# Install dependencies
npm install

# Start developing with REPL
npm run dev

# Start production
npm start

# Run unit tests
npm test

# Run continuous test mode
npm run ci

# Run ESLint
npm run lint
```

## Usage

``` bash
$ npm run dev
$ moleculer-repl: call 'math.sum' --a 2 --b 4
```
